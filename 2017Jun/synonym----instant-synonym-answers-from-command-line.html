I wrote a new tool called synonym. Check it out on Github!

synonym returns definitions, properties and synonyms of a word of interest. It is able to limit answers to the property specified (eg. noun., verb.). It features beautiful terminal colors thanks to crayons.

Motivation

Translating ideas to language is an important skill. Writing something easy and fun to read is an important skill. Thinking is important, and writing forces us to think. Therefore, it is very, very important that we write occasionally.

important counter = 4

You get the idea.

I constantly refer to thesaurus.com for synonyms when I write. I love typing in vim. And I do not want to leave the console, because I tend to not come back from a browser (major source of distraction!) from time to time.

synonym fulfills all the requirements and easy to use.

Usage

usage: synonym [-h] [-p property] [the word of interest]

positional arguments:
  The Word of Interest

optional arguments:
  -h, --help            show this help message and exit
  -p, --property        the property of interest,
                        choose from [n, v, adj, adv]

Install

pip install synonym
